# ✨ Overview

**EventSphere AI** is a modern **full-stack Event Management System** that enables organizations to efficiently manage **events, users, venues, bookings, payments, and reports** through a secure and intuitive dashboard.

Designed with scalability and usability in mind, the platform provides a seamless experience for both **administrators** and **users**, making event planning and ticket booking simple and efficient.

# ⚙️ Installation

```bash
git clone https://github.com/deepakuppala/Event-Management-System.git

cd Event-Management-System
```

Configure **application.properties**

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/eventdb
spring.datasource.username=root
spring.datasource.password=your_password
```

Run the project

```bash
mvn spring-boot:run
```

Open your browser

```
http://localhost:8080
```
