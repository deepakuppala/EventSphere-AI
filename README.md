# 🌟 About EventSphere

EventSphere is a modern full-stack event management platform that simplifies the complete event lifecycle—from creating events and managing venues to booking tickets, processing payments, generating digital tickets, and tracking attendance.

The platform provides dedicated dashboards for both administrators and users, making event organization efficient, secure, and scalable.

Whether it's a university fest, technical conference, corporate meetup, or music concert, EventSphere provides a streamlined experience for organizers and attendees.

# ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/deepakuppala/EventSphere-AI.git

cd EventSphere-AI
```

---

### Backend

Configure `application.properties`

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/event_management

spring.datasource.username=root

spring.datasource.password=your_password
```

Run Spring Boot

```bash
mvn spring-boot:run
```

---

### Frontend (React)

```bash
cd frontend

npm install

npm run dev
```

---

### Open

Backend

```
http://localhost:8080
```

Frontend

```
http://localhost:5173
```
